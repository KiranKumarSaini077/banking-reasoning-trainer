# Training Model

Core loop: **ASSESS → ISOLATE → TRAIN → COMBINE → APPLY → DIAGNOSE → RETRAIN**

V1 isolates gap calculation because distance-versus-gap confusion is a common atomic failure that contaminates larger seating arrangements.

## V1 Feedback Loop
1. Present varied surface wording.
2. Capture response time.
3. Validate response.
4. Map known distractors to likely error categories.
5. Explain the underlying operation.
6. Update deterministic mastery.
7. Persist attempt and mastery.
8. Summarize accuracy, median time, and recurring error.

## Scientific Honesty
The system uses deliberate practice, immediate explanatory feedback, error-focused practice, progressive difficulty, and persistent review data. It does not claim to increase IQ or provide clinically proven cognitive enhancement.
